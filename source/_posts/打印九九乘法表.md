### 打印九九乘法表

主要是练习python下的不同输出方式

## 代码

```python
# 用for循环打印九九乘法表

for i in range(1, 10):
    for j in range(1, 10):
        # 这里例举四种不同方式的打印方法
        # 1、
        # print("{0}*{1}={2}".format(i, j, i * j), end='\t')
        # 2 、
        # print("%d*%d=%2d" % (i, j, i * j), end='\t')
        # 3、
        # print(i, "x", j, "=", i * j, "\t", end="")
        # 4、
        print("{}*{}={}".format(i, j, i * j), end='\t')
        if i == j:
            print("")
            break

# 用while循环打印九九乘法表
row = 1
while row <= 9:
    col = 1
    while col <= row:
        print("%d*%d=%2d" % (row, col, row * col), end='\t')
        col += 1
    print("")
    row += 1
```
